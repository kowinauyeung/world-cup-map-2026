# World Cup Map 2026

An interactive web app for exploring every FIFA World Cup 2026 match by date and venue. It combines a match calendar, venue map, match details, venue time-zone information, and a day/night indicator.

The source schedule is kept in [`data.json`](./data.json). Product requirements, architecture, and implementation decisions are in [`docs/`](./docs/README.md).

## Getting started

Prerequisite: Node.js 22 or later and Yarn Classic.

```bash
yarn install
yarn dev
```

Open the local URL printed by Vite. Useful checks:

```bash
yarn lint
yarn build
```
